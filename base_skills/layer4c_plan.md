# 层4c: 中间产物链 — 实现计划

## 数据流
```
控制中枢5人 → 生成产品 → 制造站/贸易站消费 → 修正产能
```

**输出:** 各产品数量 dict

### 步骤 4c.2: 人间烟火 → 贸易效率
**消费:** `愿者上钩` (1点→+1%效率)  
**影响:** 贸易站排名可能变化 (效率直接加成)  
**实现:** 在 trading_solver 中注入 `renjian_yanhuo` 参数

### 步骤 4c.3: 人间烟火 → 制造产能 
**消费:** `稻禾厚顺秋收` (黍E2: 3点→+1%)  
**影响:** 约 +5%

### 步骤 4c.4: 热情值 → 制造产能
**消费:** `丰富工作经验` (中枢: 20点→+0.5%/+1% 贵金属制造站)  
**注意:** 这是中枢 buff，不是制造站技能 — 已在 layer2_control 部分实现

### 步骤 4c.5: 感知信息 → 思维链环 → 制造产能
**消费:** `念力`(2点→+1%), `意识实体`(1点→+1%)

### 步骤 4c.6: 木天蓼 → 制造+贸易
**消费:** `可靠随从们`(制造:1点→+1%), `艾露猫`(贸易:1点→+3%)

### 步骤 4c.7: 工程机器人链 (至简) — ✅ 已实现 (ROBOTS=64)

### 步骤 4c.8: 魔物料理 → 制造+贸易
**消费:** `意想不到的美味`(制造:1点→+1%), `熟悉的味道`(贸易:1点→+1%)
**生成:** 玛露西尔 Slot1 E2 (自身生成, 不依赖中枢)

### 步骤 4c.9: 巫术结晶链 (衍生)
**消费:** `古老巫术`(5人间烟火→1巫术结晶) → `问枯荣`(1→+2%), `逐水草`(1→+1%)
